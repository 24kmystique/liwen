GUI Game Application

TOTO

A betting game where user guess a number between 0 to 10 (inclusive) and starts with a wallet amount of $3. Before a random number is generated, user will have to place a $1 bet irregardless whether a number is entered. If user guessed the correct number, user will receive an addition of $5 in their wallet. However, if user used up all the money in his wallet, he will no longer be able to bet.
